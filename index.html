React

A JavaScript library for building user interfaces.
  • Zasada jednej odpowiedzialności zarówno dla frameworka - jedyną odpowiedzialnością Reacta jest budowanie interfejsu
  • React udostępnia komponent, który pomaga z zachowaniu Single Responsibility Principle

1. Deklaratywność

- Deklaratywność stanu - mówię jak ma wyglądać stan aplikacji, a React sam wie jak zaaplikować tą zmianę na widoku
- Deklaratywność widoku - mówię co ma być wyświetlone, a komponent sam wie jak to zrobić

2. Wydajność

Virtual DOM

Redux

Predictable state container for JavaScript apps.

Stan aplikacji jest jednym obiektem

Store

Stan jest read-only

Modyfikacja stanu następuje asynchronicznie

Akcja

Reducer

Store
